# turtly-draw
## Introduction
A simple drawing tool made with Python built-in `turtle` module.
## Requirements
- **Tk** interface package (**Tkinter**)
## Installation
```bash
git clone https://illmilo/turtly-draw.git
cd turtly-draw
```
